# Url_Shortening
## 사용기술
- Java8
- SpringBoot
- SpringData JPA
- H2 Database

## 설치 방법

(centOS 기준으로 설명 드립니다.)

* wget 설치
> sudo yum install wget

* java 설치
> sudo yum install java

* jar 파일 다운로드
> wget https://github.com/osj56/Url_Shortening/raw/master/musinsa-0.0.1-SNAPSHOT.jar

* jar 파일 실행
> java -jar musinsa-0.0.1-SNAPSHOT.jar

* 화면 출력
> http://XXX.XXX.XXX.XXX:8080/index.html
